3.系統平台與部署策略
-平台選擇與方向調整
-原本規劃開發成手機應用程式（App），但經評估後考量到後續的維護便利性與跨平台兼容性，決定改為開發基於瀏覽器的網站系統。網站平台具有較佳的擴充性與遠端維運優勢，並能更有效整合雲端資源。
-分析功能：
